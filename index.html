Background

My name is Alex Sherman and I'm a software engineer living in San Francisco. I have a background in Economics and broadcast engineering but my passion has always been computers - coding my first website in 6th grade, building my first computer in 7th grade and learning to build with C at various summer programs. Once I discovered that there was a pathway into software development for someone of my background, I found the most rigorous boot camp available and applied immediately. In my time programming I've become proficient in several languages and have built full-stack applications. I'm most fascinated with front-end technologies and enjoy challenging myself to learn new techniques.

Projects

eZ Search

Ez is a mobile application developed in collaboration between David Hu, Tom Ogasawara and myself. Currently available on the Google Play store - Ez allows users to determine the streaming location of a desired peice of media with one search. This project required my team to learn React Native in a weekend and build a cross-platform application.

Snackchat

Snackchat is a full stack social media application that uses a Ruby on Rails backend and React-Redux frontend. It allows it's users to share and discover food in their area.

Swatchbook

Swatchbook is a twelve hour project that allowed me to explore several frontend design features. It uses almost entirely html and css to showcase a range of frontend functionality.
